# Dev Minimalism

> Quantitative Trading & FinTech Engineering

## About

1x년 경력의 소프트웨어 엔지니어로, **퀀트 트레이딩 시스템**과 **핀테크 솔루션** 개발에 집중하고 있습니다.

복잡한 금융 시장을 코드로 해석하고, 데이터 기반의 투자 전략을 시스템화하는 일을 합니다.

## Expertise

**Quantitative Trading**
- 알고리즘 트레이딩 시스템 설계 및 구현
- CAN SLIM, 모멘텀, 변동성 돌파 등 다양한 전략 개발
- 기술적 지표 기반 시그널 시스템 (RSI, Williams %R, MACD)
- 백테스팅 프레임워크 구축

**AI/ML Trading**
- LightGBM, Transformer, CNN-LSTM 앙상블 모델
- 실시간 매매 시그널 생성 및 알림 시스템
- 시계열 예측 모델 개발

**Multi-Asset Coverage**
- 한국 주식 (KIS API)
- 미국 주식 (NASDAQ, NYSE)
- 암호화폐 (Upbit)

**Tech Stack**
- Backend: Java, Spring Boot, JPA/Hibernate
- Data: Python, Pandas, NumPy
- Architecture: MSA, Event-Driven, Saga Pattern
- Infrastructure: Docker, CI/CD
